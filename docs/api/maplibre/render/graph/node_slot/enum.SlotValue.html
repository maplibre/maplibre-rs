pub enum SlotValue {
    Buffer(Rc<Buffer>),
    TextureView(Rc<TextureView>),
    Sampler(Rc<Sampler>),
}
Expand description

A value passed between render Nodes. Corresponds to the SlotType specified in the RenderGraph.

Slots can have four different types of values: [Buffer], TextureView, [Sampler] and [Entity].

These values do not contain the actual render data, but only the ids to retrieve them.
