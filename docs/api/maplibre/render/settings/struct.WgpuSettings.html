    pub device_label: Option<Cow<'static, str>>,

Expand description

Provides configuration for renderer initialization. Use Device::features, Device::limits, and the WgpuAdapterInfo resource to get runtime information about the actual adapter, backend, features, and limits.

Fields§

§device_label: Option<Cow<'static, str>>§backends: Option<Backends>§power_preference: PowerPreference§features: Features

The features to ensure are enabled regardless of what the adapter/backend supports. Setting these explicitly may cause renderer initialization to fail.

§disabled_features: Option<Features>

The features to ensure are disabled regardless of what the adapter/backend supports

§limits: Limits

The imposed limits.

§constrained_limits: Option<Limits>

The constraints on limits allowed regardless of what the adapter/backend supports

§record_trace: bool

Whether a trace is recorded an stored in the current working directory
