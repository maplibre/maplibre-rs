Expand description

A color in the CIE L*a*b* color space.

Fields§

§l: ComponentTy

The L (lightness) component. Varies from 0 to 100.

§a: ComponentTy

The a component, representing green-red chroma difference.

§b: ComponentTy

The b component, representing blue-yellow chroma difference.
